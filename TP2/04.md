# Letra Ausente do Alfabeto

**Descrição da solução**  

- Criamos um conjunto (set) vazio para marcar as letras presentes.  
- Percorremos a string uma única vez:  
  - Convertemos cada caractere para minúsculo  
  - Ignoramos tudo que não for letra (espaço, número, pontuação)  
  - Adicionamos a letra ao conjunto  
- Depois comparamos com o conjunto completo do alfabeto (a–z)  
- Retornamos a primeira letra (em ordem alfabética) que não está presente

**Estrutura auxiliar utilizada**  

- Conjunto (set) → inserção e consulta em tempo médio O(1)  
- Alternativa: array booleano de tamanho 26 (a–z) também seria O(1) por acesso

## Complexidade do algoritmo

- Percorrer a string de tamanho n → **O(n)**  
- Inserir até 26 letras no conjunto → no pior caso O(26) = **O(1)**  
- Comparar com o alfabeto completo (26 letras) → **O(1)**  
- Ordenar as letras faltantes (no máximo 26) → **O(1)**

**Complexidade total**: **O(n)** no tempo  
**Espaço auxiliar**: **O(1)** (constante, pois no máximo 26 letras são armazenadas)

**Conclusão**  
O algoritmo é **linear no tamanho da entrada** (O(n)) e utiliza espaço constante O(1), sendo eficiente mesmo para strings muito longas.
